# ESP12E Mains-Powered Breakout Board (v1.0)

A custom, compact development breakout board that integrates a **Hi-Link HLK-10M05** isolated power supply module to power a **NodeMCU (ESP-12E)** directly from household **230V AC mains**.

This board is designed to act as a robust prototyping platform for IoT energy monitoring, automation, and smart infrastructure projects without needing an external DC wall adapter.
